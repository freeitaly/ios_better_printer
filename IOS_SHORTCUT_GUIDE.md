# iOS 快捷指令设置指南

在 iPhone 上创建"PDF转换打印"快捷指令，实现**一键转换文档为PDF**。

---

## ⚡ 一键安装（如有分享链接）

如果你有预配置的 iCloud 快捷指令分享链接，可以直接点击添加。

> 分享链接保存在 `PRIVATE_LINKS.md`（不提交到 Git）

---

## 🚀 快速创建（3步，1分钟）

### 步骤 1：新建快捷指令

1. 打开 iPhone 上的 **"快捷指令"** App
2. 点击右上角 **"+"**

---

### 步骤 2：添加"获取URL内容"

1. 点击 **"添加操作"** 或底部搜索框
2. 搜索 **"URL"**，选择 **"获取URL内容"**
3. 点击 **"URL"** 字段，输入：
   ```
   http://<你的服务器IP>:18080/api/convert
   ```
4. 点击 **"显示更多"**（或向下箭头）
5. 设置：
   - **方法**: `POST`
   - **请求体**: `表单`
6. 点击 **"添加新字段"** → 选择 **"文件"**
7. 设置字段：
   - **键**: `file`
   - **值**: 点击选择 **"快捷指令输入"**

---

### 步骤 3：添加"快速查看"

1. 点击底部 **"+"** 或搜索框
2. 搜索 **"快速"**，选择 **"快速查看"**
3. 输入会自动连接到上一步结果

---

### 步骤 4：保存设置

1. 点击顶部名称，改为 **"PDF转换打印"**
2. 点击名称旁 **"∨"** 或 **"ⓘ"**
3. 开启 **"在共享表单中显示"**
4. 点击 **"完成"**

---

## 📱 使用方法

1. 在**微信/文件App**中，**长按** Word/Excel/PPT 文件
2. 点击 **"共享"** 或 **"用其他应用打开"**
3. 选择 **"PDF转换打印"**
4. 等待 5-15 秒，PDF 自动打开
5. 点击右上角分享 → **"打印"**

---

## ❓ 常见问题

| 问题 | 解决方案 |
|-----|---------|
| 共享菜单找不到快捷指令 | 确认开启了"在共享表单中显示" |
| 无法连接服务器 | 检查手机网络，确认不是纯内网 |
| 转换失败 | 确认文件是 .doc/.docx/.xls/.xlsx/.ppt/.pptx |
